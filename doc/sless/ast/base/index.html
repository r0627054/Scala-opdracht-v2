    The printable trait contains all the methods which need to be implemented to print a component.

    The printable trait contains all the methods which need to be implemented to print a component. It contains a method to flat print a component or to print a component pretty. It looks like the "Compilable" trait, the difference is that these methods don't need a CSS component as parameter.
