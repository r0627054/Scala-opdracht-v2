RuleOrDeclarationComponent 

abstract class RuleOrDeclarationComponent extends BaseComponent

The rule or declaration component is an abstract class. It sub classes are either a DeclarationComponent or a RuleComponent. The class is needed for the nested style rules implementation. This is because a rule can consist of a list of both actual rules or declarations.
