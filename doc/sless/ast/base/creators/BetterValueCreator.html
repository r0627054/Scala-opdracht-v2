BetterValueCreator 

trait BetterValueCreator extends BaseAST with BetterValuesDSL

The better value creator is a creator class which implements the BetterValueDSL methods. It contains all the methods to create basic value components and property specific value components. The following value components are considered basic: Pixel, Pica, Point, mm, cm, in, em, ex, %, auto, inherit The following value components are property specific: all margin values
